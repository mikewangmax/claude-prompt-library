# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

A showcase of outputs generated by Claude from prompts in the [Anthropic Claude Prompt Library](https://platform.claude.com/docs/en/resources/prompt-library/library). Each folder holds a `prompt.md` (the exact input given to Claude) alongside the generated output file(s).

## Structure Convention

Every demo folder follows this pattern:

```
DemoName/
├── prompt.md      # The prompt(s) sent to Claude — do not modify these
└── <output>       # The Claude-generated output (HTML, .md, .gs, etc.)
```

Adding a new demo means creating a new top-level folder with this same structure.

## GitHub Pages

Live HTML demos are served via GitHub Pages from the `main` branch root:

```
https://mikewangmax.github.io/claude-prompt-library/<FolderName>/<file.html>
```

When adding or renaming HTML output files, update the corresponding **Live demo** link in `README.md`.

## Git / GitHub Operations

```bash
# Publish changes
git add <files> && git commit -m "..." && git push

# Repo and Pages management (approved permissions)
gh repo edit mikewangmax/claude-prompt-library --visibility <public|private>
gh api repos/mikewangmax/claude-prompt-library/pages --method POST -f source[branch]=main -f source[path]=/
```

Allowed `gh` and `git` operations are explicitly permitted in `.claude/settings.local.json`.
